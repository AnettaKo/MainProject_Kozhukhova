"My Wardrobe" is an application to record closes.

A user can work with wardrobe articles: add new one, change, delete articles, and make reports with selections.
All information is inputted in a console. The program use classificators and notes for quick input.
Reports are displayed in a browser. A "wardrobe.csv" file is used to save information.
Information is read from the file when the program starts and written to the file before exiting the program.

Requirements:
Python 3.9
Install the packages specified in the file "requirements.txt"

Usage:
Run "main.ru"
